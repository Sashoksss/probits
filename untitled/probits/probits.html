Настоящее приложение позволяет произвести расчет значений летальных доз (ЛД50 ± S, ЛД84, ЛД16) на основании данных, полученных в остром эксперименте на лабораторных животных по методу Финни.

В каждой строке вводятся данные для одной группы животных.

В строке под номером 1 всегда вводятся данные для группы животных с процентом гибели равным 0.

Далее вводятся данные для групп с процентом гибели более 0 и менее 100 по нарастающей, от меньшего к большему

В последней строке вводятся данные для группы животным с процентом гибели 100

Число экспериментальных групп должно соответствовать числу заполненных строк!

Для ввода дробных значений в поле «доза» в качестве разделителя используется либо точка либо запятая либо возможно сочетание этих знаков. (будьте внимательны - дозы испытуемых веществ должны выражаться в одних единицах (концентрациях)!!!)
